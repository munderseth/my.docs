## Sitemap

Reference - https://github.com/jekyll/jekyll-sitemap

### Exclusions
Exclude specific [page](https://github.com/jekyll/jekyll-sitemap#exclusions) use the front matter section:
```
sitemap: false
```

## Comment out

Use following syntax:
```
{% comment %}
...
{% endcomment %}
```

Reference [Stackoverflow](https://stackoverflow.com/questions/27975757/comment-out-an-include-statement-inside-an-html-file-using-jekyll).
